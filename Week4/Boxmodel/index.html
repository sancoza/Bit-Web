3. Exercise CSS display

Block elements take the available full width . Those elements always start with a new line (p,h1, hr).

  • display:block;
  • display:block;
  • display:block;
Inline elements take only the required width. Those elements do not start with a new line (a, b, i, img).
display:inline; display:inline; display:inline;

Inline-block is placed as an inline element (on the same line as adjacent content), but it behaves as a block element.
One common use for display: inline-block is to display list items horizontally instead of vertically.

  • ul li display:inline-block
  • ul li display:inline-block
  • ul li display:inline-block
